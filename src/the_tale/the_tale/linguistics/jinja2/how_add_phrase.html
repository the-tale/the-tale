Чтобы фраза появилась в игре, должна произойти следующая цепь событий:

  1. Игрок добавляет шаблон фразы выбранного типа (подробнее о шаблонах).
  2. Если шаблон использует слова, которых ещё нет в словаре, игрок добавляет необходимые слова в словарь.
  3. Модератор проверяет добавленные слова и, если они проходят проверку, вносит их в игру.
  4. С этого момента шаблоны фраз «видят» слова.
  5. Игрок исправляет ошибки в шаблоне, если такие есть.
  6. С этого момента шаблон помечается как чистый.
  7. Если шаблон подходит по стилю и по тематике, то модератор вносит его в игру.
  8. И слова, и шаблоны могут быть удалены модератором без объяснения причин в любое время.

При создании шаблона фразы, кроме самого шаблона необходимо указывать несколько проверочных фраз. Проверочная фраза — это ожидаемый текст после подставновки всех переменных. Для каждого типа фразы установлен свой набор предопредлённых переменных, покрывающих основные варианты использования шаблона. Это необходимо, чтобы свести к минимуму опечатки в формате шаблона и убедиться, что все изменяемые слова размечены корректно.

Игроки могут изменять уже созданные фразы и слова, при этом:

  • При изменении фразы или слова, находящихся в игре, создаётся их непроверенная копия, привязанная к оригиналам.
  • Непроверенные фразы и слова могут редактировать только их авторы, редакторы и модераторы.
  • Если Вы хотите отредактировать чужое непроверенное слово или фразу, дождитесь, пока модератор внесёт их в игру или удалит.
  • Или свяжитесь с их автором.
  • Непроверенную копию слова модератор либо удалит, либо заменит ей оригинал.
  • Непроверенную копию фразы модератор либо удалит, либо заменит ей оригинал, либо «открепит» и сделает отдельной фразой.

Для каждой фразы и слова хранятся списки игроков, чьи изменения были приняты в игру. Могущество начисляется за каждую принятую фразу (и слово) и делится по следующим правилам:

  1. Могущество считается отдельно для игроков и для модераторов;
  2. Автор принятой фразы (или слова) получает фиксированное количество могущества;
  3. Редакторы принятой фразы (или слова) делят поровну между собой фиксированное количество могущества.

Требования к фразам

  1. Фразы должны соответствовать миру игры. Помните, что даже юмора может быть слишком много.
  2. Фразы должны соответствовать миру игры.
  3. ФРАЗЫ ДОЛЖНЫ СООТВЕТСТВОВАТЬ МИРУ ИГРЫ.
  4. Фраза должна соответствовать своему типу.
  5. Не допускаются оскорбления и ненормативная лексика.
  6. Будьте оригинальны!

Контекст использования фразы

Для каждой фразы можно указать список дополнительных требований к переменным, с которыми она используется. Например, можно указать, что фраза используется только для героев-женщин или для определённого типа монстра (или даже для конкретного типа). Фраза с таким ограничением будет использована только, если переменная отвечает указанным требованиям.

Прочие ограничения

  • Мы используем букву Ё, в новых словах и фразах использовать её необходимо, старые необходимо исправлять с учётом использования этой буквы.
  • На текущий момент, при сравнении с проверочными фразами, буквы Е и Ё считаются равными, чтобы облегчить исправление слов на использование буквы ё.
  • Новое слово пройдёт проверку модератора только, если оно уже используется в одной из фраз-кандидаток. Слова без фраз проверку не пройдут.
